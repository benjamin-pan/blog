如何使用

  1. 克隆项目
    git clone git@github.com:benjamin-pan/blogs.git

  2. 依赖安装
    npm install

  3. 新建博客文章
    node_modules/hexo/bin/hexo new "文章标题"

  4. 本地预览
    node_modules/hexo/bin/hexo server

  5. 全局安装 hexo，之后可以用 hexo 代替node_modules/hexo/bin/hexo
    npm install hexo-cli -g

  6. 提交自己分支，并发起merge request
